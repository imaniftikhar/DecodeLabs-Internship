RULE-BASED CHATBOT

Project Overview
This project is a simple rule-based chatbot developed in Python. The chatbot interacts with users through the command line and responds to predefined inputs using if-else statements and decision-making logic. It demonstrates the basic concepts of conversational AI without using machine learning or external libraries.

Features
* Handles greetings such as Hi, Hello, and Hey.
* Recognizes exit commands like Bye, Exit, and Quit.
* Responds to questions about its name.
* Replies to basic emotions (happy, sad, tired).
* Displays the current system time.
* Answers simple Python-related questions.
* Tells a programming joke.
* Maintains a continuous conversation until the user exits.
* Uses nested conditions for smarter responses.
* Includes a friendly chatbot personality.

Technologies Used
* Python 3
* If-Else Statements
* While Loop
* Decision-Making Logic
* Basic String Handling

How to Run
1. Make sure Python 3 is installed on your computer.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the following command:

   python rulebasedchatbot.py

Example Conversation
Kathie: Hello! I am Kathie, your friendly chatbot.
Type 'bye' anytime to end the conversation.

You: hi
Kathie: Hello! Nice to meet you.

You: how are you
Kathie: I'm doing great! Thanks for asking.

You: I am happy
Kathie: That's wonderful! Keep smiling!

You: tell me a joke
Kathie: Why do programmers prefer dark mode?
Kathie: Because light attracts bugs!

You: bye
Kathie: Goodbye! Have a wonderful day.
